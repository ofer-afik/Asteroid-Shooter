# Asteroid-Shooter
A terminal-based game that uses the python curses module. The name describes itself :)

### To run:
#### Requirements:
- Python 3.x.x
- Terminal or command prompt to run
- `windows-curses` [package](https://pypi.org/project/windows-curses/) installed **(Windows only)**

#### Linux/macOS
1. Open a terminal
2. Navigate to the directory where the `game.py` file is saved:
`cd /path/to/the/folder`
4. Run the script:
`python3 game.py`
5. Enjoy!

#### Windows
1. Open Command Prompt
2. Make sure you have `windows-curses` installed (Link in the requirements)
3. Navigate to the folder:
`cd C:\path\to\your\folder`
4. Run the script:
`python game.py`
5. Enjoy!
